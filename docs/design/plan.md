# План создания макетов в Figma

Документ описывает **только** работу в Figma по этому проекту. Требования к страницам и блокам — из [ТЗ в Google Docs](https://docs.google.com/document/d/1--N5B8rCHCvRSyfK8uuXMWmFgGNsZGkyetUHX1KOeHc/edit?tab=t.0). Рабочий файл: [Figma — Untitled](https://www.figma.com/design/SIOFqk6OdgyZCgq7Hgovd5/Untitled?t=KH7MP7x11F1j3yvy-0). Автоматизация в файле — через MCP **figma-mcp-go** (см. [AGENTS.md](../../AGENTS.md)); при недоступности MCP — не имитировать правки в Figma, зафиксировать в чате и при необходимости перенести шаги вручную или скриптом в репозитории.

Выполняй шаги **по порядку**. После каждого шага можно зафиксировать состояние (версия в Figma / комментарий в задаче).

## Референсы по стилю (ориентир, не брендбук)

Перед финализацией цветов и компонентов полезно просмотреть **русскоязычные** кейсы (интерфейс и контент в макетах на русском, либо проект явно для РФ/СНГ): спокойная палитра, читаемая типографика для кириллицы, доверие, понятные CTA и путь к записи. **Не копируй** чужие логотипы и фирменные решения; используй ссылки как настроение и идеи композиции.
