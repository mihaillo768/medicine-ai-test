# Прототип сайта медицинского центра

Клиентский прототип для условного медцентра (взрослые пациенты): услуги, врачи, запись, контакты, блог и др. **Backend и CMS в репозитории нет** — только статический фронтенд и вспомогательные материалы.

## Структура репозитория

| Путь | Содержимое |
|------|------------|
| `frontend/` | Приложение **Vue 3**, Vite, Vue Router — то, что деплоится (например на Vercel). |
| `docs/` | Планы и заметки: фронтенд, дизайн, IA. |
| `tools/figma-mcp-go/` | Локальный форк MCP для работы с Figma из Cursor (опционально для разработки). |
| `AGENTS.md` | Правила для ИИ-агентов и ссылки на ТЗ / Figma. |

## Как развернуть у себя

**Требования:** Node.js **18+**, npm.

1. Клонировать репозиторий и перейти в каталог фронтенда:
   ```bash
   git clone git@github.com:mihaillo768/medicine-ai-test.git
   cd medicine-ai-test/frontend
   ```
2. Установить зависимости и запустить dev-сервер:
   ```bash
   npm install
   npm run dev
   ```
3. Открыть в браузере адрес, который выведет Vite (обычно `http://localhost:5173`).

**Production-сборка локально:** `npm run build`, затем `npm run preview` — проверка собранного `dist/`.

Подробности по командам и структуре `src/` — в [frontend/README.md](frontend/README.md). План вёрстки: [docs/frontend/plan.md](docs/frontend/plan.md).
